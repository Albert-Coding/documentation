### `--run-id`

Show the History of a [Workflow Execution](/concepts/what-is-a-workflow-execution) by specifying a [Run Id](/concepts/what-is-a-run-id).

Alias: `--rid`

**Example**

```bash
tctl workflow <command> --run-id <value>
```
